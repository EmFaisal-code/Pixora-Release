# Pixora — The Ultimate TikTok Upload Booster & Analyzer 🚀

Ekstensi Chrome eksklusif untuk **Affiliate**, **Kreator**, dan **Growth Hacker** TikTok. 

## 🔥 Fitur Utama

- **Upload Boost (Anti-Compress)** — Secara otomatis mencegah TikTok melakukan kompresi brutal pada videomu saat di-upload via Web.
- **Prepare Video (Bypass MD5)** — Mesin FFmpeg lokal yang memodifikasi *metadata* (MD5, Frame Rate) file MP4 agar algoritma TikTok menganggapnya sebagai video original 100% (Anti Re-upload).
- **Profile Spy & Analytics 2.0** — Mengintip di balik layar akun kompetitor:
  - 📈 Kalkulator *Engagement Rate* (ER) & *Loyalty Value* (LV).
  - ⏰ Deteksi Jam Upload Terbaik (*Best Hour*).
  - 🔥 Ekstraksi Tagar (*Top Hashtags*) dan Musik (*Trending Sounds*).
- **FYP Predictor** — Memindai skor metrik video secara *real-time* (Tier 1 Views) untuk memprediksi potensi FYP.
- **Download Media** — Unduh video HD tanpa *watermark* dan audio MP3 langsung dari *feed*.
- **Menu Spy**: Buka profil TikTok siapapun, lalu buka ekstensi Pixora dan lihat tab Status Akun.
- **Menu FYP**: Klik Check FYP saat memutar video di *feed* untuk melihat skor FYP-nya.

## 🛠 Instalasi (Manual Load)

Karena ekstensi ini tidak tersedia di Chrome Web Store (fiturnya terlalu 'underground'), kamu harus menginstalnya secara manual:

1. Klik tombol **Code** hijau di repo ini, lalu pilih **Download ZIP**.
2. Ekstrak file ZIP tersebut ke sebuah folder.
3. Buka Google Chrome dan ketik `chrome://extensions` di kolom URL.
4. Aktifkan **Developer mode** (saklar di pojok kanan atas).
5. Klik tombol **Load unpacked** (di kiri atas) → pilih folder yang tadi kamu ekstrak.
6. Icon Pixora akan muncul di *toolbar* Chrome kamu. 

## 📖 Cara Pakai

1. Buka [TikTok](https://www.tiktok.com) dan pastikan kamu sudah Login ke akunmu.
2. Klik icon Pixora di *toolbar*.
3. Pilih methode mana yang ingin digunakan
3. **Menu Prepare**: Drag & drop video kamu ke dalam kotak untuk mencuci *metadata*-nya atau langsung drop ke tiktok.
